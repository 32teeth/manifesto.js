manifesto.js github

javascript library for offline cache manifest management

whats it for

manifesto.js is a application cache (appcache) management utility that allows you to view the state of the current application manifest and it's files, add callbacks to the 8 event states of the browsers appcache manager;

  • checking
  • error
  • noupdate
  • downloading
  • progress
  • updateready
  • cached
  • obsolete
  • reload

on first load you will notice messsaging for checking, downloading, progress and updateready events.

on page reload you will have checking and no update if the manifest hasn't changed
